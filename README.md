# saltstack-cli-golang-hello-world

## Description
A POC project for go; ran as an
uncompiled script. Built by salt.

## Tech stack
- salt
- golang

## Docker stack
- docker-cli
- ubuntu:latest

## Requirements
Docker desktop must be installed and the application
being called must be linux compatiple.

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
